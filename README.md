# Hidden Costs of AI Website

This is a static website for a community service project about how AI data centers and computing infrastructure can increase demand for water, electricity, land, and natural resources.

## Files

- `index.html` - website structure and content
- `styles.css` - design and responsive layout
- `script.js` - mobile menu and pledge form behavior

## How to edit

Open the files in any code editor. Replace:

- `[Your Name]`
- Source placeholders
- Any project-specific details your teacher requires

## How to publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Go to repository Settings.
4. Open Pages.
5. Set source to `main` branch.
6. Save and wait for the public website link.
